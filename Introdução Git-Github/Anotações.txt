-> Comandos Git/Repositório
git clone link = para clonar o repositorio a sua pasta;
git status = mostra a situação atual que o repositório se encontra;
git add . ou git add -A = revela a pasta do arquivo ao repositorio local;
git push = adiciona ao repositorio Github.
